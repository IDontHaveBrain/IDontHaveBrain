## 💼 소개
분산 시스템 설계, 복잡한 기술적 문제 해결에 전문성을 가진 백엔드 개발자입니다. 새로운 기술적 도전을 적극적으로 수용하며, 복잡한 레거시 시스템을 현대적인 아키텍처로 전환하고, 지속적 통합/배포 파이프라인을 구축하는 경험을 쌓아왔습니다. 기술적 도전을 즐기며 가독성, 확장성, 유지보수성을 모두 고려한 소프트웨어 개발에 주력합니다.
